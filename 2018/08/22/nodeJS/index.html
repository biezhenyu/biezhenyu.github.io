process - 进程

process 对象是一个全局变量，它提供当前 Node.js 进程的有关信息，以及控制当前 Node.js 进程。 因为是全局变量，所以无需使用 require()。

argv

argv属性返回一个数组，这个数组包含了启动Node.js进程时的命令行参数。默认情况下返回两个元素，第一个是node安装的路径，第二个是当前执行文件的路径。

nextTick

将 callback 添加到”next tick 队列”。 一旦当前事件轮询队列的任务全部完成，在next tick队列中的所有callbacks会被依次调用。

    实例类似整数数组，但是大小是固定的，创建后无法被调整

length

toString

fill

接受四个参数

  • value 用来填充 buf 的值。
  • offset 起始位置
  • end 结束为止
