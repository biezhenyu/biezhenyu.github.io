不是值的值

  • undefined指从未赋值
  • null指曾经赋过值，但是目前没有值

null是一个特殊关键字，不能当做变量来赋值和使用，undefined确实一个标识符，可以当做变量来赋值和使

特殊数字

NaN

如果数学运算的操作数不是数字类型，就无法返回一个有效的数字

NaN准确的理解为，“无效数值”，“失败数值”，或者“坏数值”更准确些

NaN是一个“警戒值”，用于指出数字类型中错误情况

NaN同是也是一个特殊值，它和自身并不相等

JSON字符串化

JSON字符串化并非强制类型的转化，对于大多数简单的值来说，JSON字符串化和toString的效果基本相同，只不过序列化的结果总是字符串
